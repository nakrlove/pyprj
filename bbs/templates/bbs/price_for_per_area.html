모델 조회 데모

상단에서 모델을 선택하고 조회를 클릭하세요. 선택된 모델의 요약 정보와 예시 결과가 이 영역에 표시됩니다.

아직 조회 전입니다. 원하는 항목을 선택한 후 조회를 누르세요.

백엔드 연동 시 /api/search?model=모델명 과 같이 연결하면 됩니다. 예시 코드는 하단 참고.

백엔드 연동 예시 (fetch)

fetch(`/api/search?model=${'{선택된_모델'}`})
  .then(r => r.json())
  .then(data => renderResult(data))
  .catch(err => showError(err));
